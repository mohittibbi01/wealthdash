नोट: यह काम complex है। सभी inline JS/CSS को nonce देना होगा। यह सबसे last में करो — बाकी सारे काम पहले complete करो।

Main DevVault Pro ki sab files de raha hoon: auth.php, index.php, admin.php, project_form.php, findings.php, workorders.php, reports.php, export.php, sr.php, change_password.php Kaam karna hai — Nonce-based Content Security Policy implement karo (unsafe-inline remove karo): 1. auth.php mein: - Har request ke liye ek unique nonce generate karo: `$csp_nonce = base64_encode(random_bytes(16));` - Session mein store karo: `$_SESSION['csp_nonce'] = $csp_nonce;` - CSP header update karo: `script-src 'self' 'nonce-{$csp_nonce}'` `style-src 'self' 'nonce-{$csp_nonce}'` (fonts.googleapis.com agar self-hosted na ho to include karo) - 'unsafe-inline' DONO jagah se remove karo 2. Har PHP file mein: - `$nonce = $_SESSION['csp_nonce'] ?? '';` se nonce fetch karo - Sab `
