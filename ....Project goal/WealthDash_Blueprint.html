Sample Code Snippets

config/db.php — PDO Connection

// config/db.php
<?php
define('DB_HOST', 'localhost');
define('DB_NAME', 'wealthdash');
define('DB_USER', 'root');
define('DB_PASS', 'your_password');

try {
  $pdo = new PDO("mysql:host=".DB_HOST.";dbname=".DB_NAME, DB_USER, DB_PASS);
  $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch(PDOException $e) {
  die("Connection failed: " . $e->getMessage());
}

Sample MySQL — mutual_funds table

CREATE TABLE mutual_funds (
  id             INT AUTO_INCREMENT PRIMARY KEY,
  user_id        INT NOT NULL,
  scheme_name    VARCHAR(200) NOT NULL,
  folio_no       VARCHAR(50),
  units          DECIMAL(12,4) DEFAULT 0,
  avg_nav        DECIMAL(10,2) DEFAULT 0,
  current_nav    DECIMAL(10,2) DEFAULT 0,
  sip_amount     DECIMAL(10,2) DEFAULT 0,
  sip_date       TINYINT,
  start_date     DATE,
  created_at     TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

Security Checklist

🔒 Auth & Sessions

Password hashing with password_hash() · CSRF tokens · Secure session config · Remember-me with DB tokens

🛡️ SQL & Input

PDO prepared statements everywhere · htmlspecialchars() on output · Input validation & sanitisation

📁 File & Server

config/ outside public root · .htaccess deny on sensitive dirs · HTTPS enforced · Error display off in prod

🔑 API Keys

Store in .env or config (not in code) · Add .gitignore for config/api_keys.php · Rate limit API calls
